# 代码块语言标识测试

## JavaScript 示例

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

## Python 示例

```python
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```

